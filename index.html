OhmniViewTM - An All-Around Mobile Toolbox for Electronic Designs


OhmniViewTM

OhmniView offers a variety of features for electronic designers and hobbyists to build and analyze circuits on mobile devices. Using on-device sensors including microphone and speaker, OhmniView can automatically connect input/output signals for the built-in analog simulator. The app's interactive dashboard also facilitates easier remote collaborations and design reviews, while other sharing functions enable teams to easily share design resources such as component symbols or device models.



Features

Versatility

  • CKTDraw - Schematic editor with a built-in analog simulator
  • WaveBox - Graphical scope with drag-n-drop controls
  • GDSView - GDS-II viewer for layout visualization / measurement
  • RFGear - Interactive SmithChart / FFT-based noise analyses

Flexibility and Productivity

  • Design data imported via share and stays on device off-line
  • Server-based large data import option over VPN
  • Simulator integrated with on-device microphone / speaker
  • Schematic digitizer converts photos into schematics


Data Compatibility

  • GDS-II / various waveform formats / direct gzipped data import
  • Content export in PDF / SVG / PNG formats for documentation
  • ASY symbol file import / ASC schematic file import
  • Built-in SPICE models for Diode/Bipolar/JFet/MosFet devices

Remote Collaborations

  • Share designs easily via email, AirDropTM or cloud drives
  • Share reusable resources : symbols / device models / layer maps
  • Interactive dashboard of design summary for reveiw or discussion
  • Check designs anywhere off-line on mobile phones/pads
